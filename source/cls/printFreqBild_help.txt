.. code-block:: text 

    usage: printFreqBild.py [-h] [-m 1,2+3,4] [-s max displacement] [-r]                            [-c BILD 1 color [BILD 2 color ...]]                            input file        print Chimera bild file with vectors for the specified normal modes to std out        positional arguments:      input file            a frequency job output file        options:      -h, --help            show this help message and exit      -m 1,2+3,4, --mode 1,2+3,4                            mode(s) to print (1-indexed)                            Default is to print all imaginary modes separately                            - comma (,) delimited modes will be printed separately                            - plus (+) delimited modes will be combined      -s max displacement, --scale max displacement                            scale the longest vector to be this many Angstroms long                            default is 1.5                            may be delimited in accordance with the --mode option      -r, --remove-mass     remove mass-weighting from normal modes      -c BILD 1 color [BILD 2 color ...], --color BILD 1 color [BILD 2 color ...]                            color of vectors
